# DevX

DevX is an library for common functions and utilities used in the development.

## Installation

To install DevX, run the following command:

```shell
go get github.com/xBlaz3kx/DevX@latest
```

## Usage

```go
package main

import (
	"github.com/xBlaz3kx/DevX"
)

func main() {
	// Your code here
}
```
